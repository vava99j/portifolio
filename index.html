App Tec-Hor

Aplicativo mobile feito rem React-Native para o cliente obter horarios para cuidar de plantas(personalizado para cada planta) e tambem tem a função de usar nossa estufa, por via app de qualquer lugar do mundo o cliente tera como programar a horta

Detalhes Técnicos

Tecnologias usadas: Node.js, React-Native, API GEMINI e API PLANT.ID

Trageto tecnico da informação:
User faz login, Cliente tira foto da planta que ele tem, Plant.ID analisa e atraves de prompt personalizado usa o gemini para obter cuidados da planta e envia para o BD e na aba login é feita a sincronização com a estufa e a planta desejada.

Detalhes Técnicos
